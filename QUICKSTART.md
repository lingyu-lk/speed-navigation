# 快速开始指南

## 🚀 立即使用

### 方法一：直接打开（推荐用于快速体验）
1. 双击 `index.html` 文件
2. 会用默认浏览器打开
3. 所有功能已可正常使用！

### 方法二：本地服务器（推荐用于开发）

#### 使用 Python（如果已安装）
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后访问：http://localhost:8000

#### 使用 Node.js（如果已安装）
```bash
# 安装 serve
npm install -g serve

# 运行服务器
serve
```

#### 使用 VS Code
1. 安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## ✨ 主要功能

### 🔍 搜索
- **快捷键**：`Ctrl/Cmd + K` 聚焦搜索框
- **支持**：中文、英文、拼音
- **示例**：
  - 输入 "bili" 找到"哔哩哔哩"
  - 输入 "blbl" 也能找到
  - 输入 "视频" 找到所有视频相关网站

### 🌙 夜间模式
- **方法 1**：点击右上角按钮
- **方法 2**：按 `Ctrl/Cmd + D`
- **自动保存**：下次打开自动应用

### ⭐ 收藏网站
1. 在任意网站卡片上**点击右键**
2. 卡片会显示星标 ⭐
3. 再次右键点击取消收藏

### 📊 历史记录
- 自动记录访问的网站
- 保存访问次数
- 本地存储，隐私安全

## ⌨️ 键盘快捷键

## 🔧 自定义

### 添加网站
编辑 `data/sites.json` 或 `js/sites-data.js`：

```json
{
  "name": "你的网站",
  "url": "https://example.com",
  "icon": "🌐",
  "description": "网站描述",
  "tag": "标签"
}
```

保存后刷新页面即可。

### 修改颜色
编辑 `css/style.css` 中的 CSS 变量：

```css
:root {
    --accent-color: #你的颜色;
}
```

## 🐛 故障排除

### 问题：页面显示空白
**解决方案**：
1. 检查浏览器控制台（F12）是否有错误
2. 确保所有文件在同一目录下
3. 尝试刷新页面（Ctrl + Shift + R）

### 问题：搜索不工作
**解决方案**：
1. 确保 `js/pinyin.js` 已加载
2. 检查浏览器控制台是否有错误
3. 清除浏览器缓存

### 问题：夜间模式不保存
**解决方案**：
1. 检查浏览器是否启用了隐私模式
2. 确保浏览器允许 localStorage
3. 检查浏览器设置中的 Cookie 设置

### 问题：收藏/历史记录丢失
**解决方案**：
- 数据保存在 localStorage，可能因为：
  - 清除了浏览器数据
  - 使用了无痕模式
  - 更换了浏览器或设备

### 问题：修改数据不生效
**解决方案**：
1. 清除浏览器缓存
2. 强制刷新（Ctrl + Shift + R）
3. 检查 JSON 格式是否正确
